## Python make a simple Function

Let's make a simple function in Python

```python
def add(a,b):
  return a + b
```

We can new call this function passing two parameters.

```
add(2,3)

```

and get back

```
5

```
Now we can make a function for sub. with two parameters

```python
def sub(a,b):
  return a - b
```
